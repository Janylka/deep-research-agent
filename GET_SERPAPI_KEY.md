# How to Get SerpAPI Key (Free)

## Quick Steps

1. **Go to SerpAPI**
   - Visit: https://serpapi.com/users/sign_up

2. **Sign Up (Free)**
   - Enter your email
   - Create password
   - Verify email

3. **Get API Key**
   - After login, go to: https://serpapi.com/manage-api-key
   - Copy your API key (starts with a long string)

4. **Free Tier Includes:**
   - ✅ 100 searches per month
   - ✅ All search engines
   - ✅ No credit card required
   - ✅ Perfect for testing and development

## After Getting the Key

Create `.env` file in backend folder:

```bash
cd backend
```

Create file `backend/.env` with:

```env
SERP_API_KEY=your_serpapi_key_here
ANTHROPIC_API_KEY=your_anthropic_key_here
```

Then you'll be ready to run the project!
