# Push to GitHub - Quick Guide

## Step 1: Create Repository on GitHub

1. Go to: https://github.com/new
2. Repository name: `deep-research-agent`
3. Description: `AI-powered research agent with SerpAPI, JinaAI, and Claude`
4. Choose: Public or Private
5. **IMPORTANT**: DO NOT check any boxes (no README, no .gitignore, no license)
6. Click "Create repository"

## Step 2: Push Your Code

After creating the repository, run:

```bash
git push -u origin master
```

That's it! The remote is already configured to https://github.com/Janylka/deep-research-agent.git

## Step 3: Verify

Open https://github.com/Janylka/deep-research-agent to see your code!
